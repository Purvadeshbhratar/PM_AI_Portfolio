# Purva — AI Product Management Portfolio

This repository is Purva's AI Product Management (AI PM) portfolio. It showcases projects, frameworks, and case studies demonstrating product strategy, AI use-case design, data-informed roadmaps, and collaboration with engineering and research teams.

## About

- **Role focus:** AI Product Management — aligning AI capabilities with user value, ethical considerations, and scalable delivery.
- **Goal of this repo:** Present project summaries, artifacts, and links that illustrate PM thinking for AI-driven products.

## Contents

- Project write-ups with problem statements, metrics, and launch outcomes.
- Design and research artifacts (user journeys, hypotheses, experiment plans).
- Technical summaries (data needs, model tradeoffs, evaluation metrics).
- Roadmaps and prioritization examples.

## Skills & Themes

- Product strategy and discovery for AI features
- Defining success metrics and experiment design
- Cross-functional leadership with engineering & ML research
- Responsible AI: fairness, privacy, and transparency

## How to use this repo

1. Browse the `projects/` folder for individual case studies.
2. Read artefacts in `artifacts/` for templates and examples.
3. Open `README.md` in each project for details, results, and links to demos.

## Push to GitHub

To publish this repo to your GitHub account, run these commands locally (replace the remote URL):

```bash
git remote add origin https://github.com/<your-username>/Purva_Portfolio_PM.git
git push -u origin main
```
